# 五子棋游戏改进说明

## 原始代码分析

原始代码 (`gomoku.py`) 是一个基础的五子棋游戏，包含以下功能：
1. 15x15棋盘
2. 双人对战模式
3. 基本的胜负判定算法
4. 输入验证
5. 美观的棋盘显示

## 改进内容

### 1. AI对战模式（电脑玩家）
- **新增AIPlayer类**：支持三种难度级别
  - 简单难度：随机落子
  - 中等难度：优先选择中心区域
  - 困难难度：使用评估函数，有基本的攻防策略
- **AI算法实现**：
  - 检查立即获胜的机会
  - 防守对手的获胜机会
  - 位置评估函数（中心位置得分更高，连线长度得分）

### 2. 游戏难度选择
- **三种难度级别**：
  - 简单：适合新手，AI行为可预测
  - 中等：有一定策略，优先控制中心
  - 困难：使用评估函数，有攻防意识
- **难度配置**：在人机对战和机机对战模式中可分别设置难度

### 3. 游戏记录功能
- **GameSaver类**：管理游戏保存/加载
  - 支持手动保存游戏
  - 支持从保存点加载游戏
  - 自动生成时间戳保存文件
- **保存格式**：使用JSON格式保存游戏状态
- **保存内容**：棋盘状态、玩家信息、当前回合、游戏模式等

### 4. 游戏统计功能
- **GameStatistics类**：管理游戏统计信息
  - 记录总游戏数
  - 各模式胜负统计（双人对战、人机对战、机机对战）
  - 胜率、平局率计算
  - 游戏历史记录（最近5场）
- **自动保存**：统计信息自动保存到JSON文件
- **统计显示**：清晰的统计信息展示界面

### 5. 改进的用户界面和菜单系统
- **主菜单系统**：
  - 7个主要选项：三种游戏模式、加载游戏、查看统计、查看保存、退出
  - 直观的菜单导航
- **游戏内菜单**：
  - 继续游戏、保存游戏、认输、返回主菜单
  - 随时可访问，不中断游戏体验
- **增强的游戏信息显示**：
  - 显示当前回合玩家
  - 显示已落子数
  - 显示游戏模式和难度级别
- **新增命令**：
  - `menu`：打开游戏内菜单
  - `undo`：撤销上一步（仅限双人对战）
  - 保持原有的`quit`/`q`/`exit`命令

### 6. 其他改进
- **撤销功能**：双人对战模式下可撤销上一步棋
- **玩家基类**：统一的Player基类，支持人类玩家和AI玩家
- **棋盘增强**：
  - 落子历史记录
  - 获取所有空位置的方法
  - 改进的棋盘显示
- **错误处理**：增强的输入验证和错误提示
- **代码结构**：模块化设计，遵循单一职责原则

```
├── 菜单系统

```

### AI算法实现

#### 困难难度评估函数
1. **中心位置得分**：距离中心越近得分越高
2. **连线评估**：
   - 二连：5分
   - 三连：20分
   - 四连：50分
   - 五连：100分（立即获胜）
3. **攻防平衡**：
   - 进攻得分权重：10
   - 防守得分权重：8

#### 游戏流程优化
1. **状态管理**：清晰的状态转换
2. **用户交互**：友好的提示和错误信息
3. **性能考虑**：AI思考时间模拟，避免过快响应
