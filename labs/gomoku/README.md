# 五子棋游戏增强版

这是一个功能丰富的五子棋游戏，基于原始版本进行了全面增强。

## 新增功能

### 1. AI对战模式
- **简单AI**: 随机落子
- **中等AI**: 优先选择中心区域
- **困难AI**: 使用评估函数，有基本的攻防策略

### 2. 游戏模式
- **双人对战模式**: 两个人轮流下棋
- **人机对战模式**: 玩家与AI对战，可选择执棋方和难度
- **机机对战模式**: 两个AI对战，可分别设置难度

### 3. 游戏难度选择
- **简单**: AI随机落子，适合新手
- **中等**: AI优先选择中心区域，有一定策略
- **困难**: AI使用评估函数，有攻防意识

### 4. 游戏记录功能
- **自动保存统计**: 记录所有游戏结果
- **手动保存游戏**: 游戏过程中可随时保存
- **加载游戏**: 从保存点继续游戏
- **查看历史**: 查看最近5场游戏记录

### 5. 游戏统计
- **总游戏数统计**
- **各模式胜率统计**
- **游戏历史记录**
- **胜率、平局率计算**

### 6. 改进的用户界面
- **主菜单系统**: 直观的模式选择
- **游戏内菜单**: 保存、认输、返回主菜单
- **撤销功能**: 双人对战模式下可撤销上一步
- **实时信息显示**: 显示当前回合、已落子数、游戏模式等

## 游戏操作说明

### 主菜单选项
1. **双人对战模式**: 两个人轮流下棋
2. **人机对战模式**: 与AI对战，可选择难度和执棋方
3. **机机对战模式**: 观看两个AI对战
4. **加载游戏**: 从保存点继续游戏
5. **查看游戏统计**: 查看游戏历史记录和统计信息
6. **查看保存的游戏**: 列出所有保存的游戏
7. **退出游戏**: 退出程序

### 游戏内操作
- **输入坐标**: 输入"行 列"格式的坐标（如"7 7"）
- **打开菜单**: 输入"menu"打开游戏内菜单
- **撤销上一步**: 输入"undo"（仅限双人对战模式）
- **认输退出**: 输入"quit"、"q"或"exit"

### 游戏内菜单
1. **继续游戏**: 返回游戏
2. **保存游戏**: 保存当前游戏状态
3. **认输**: 当前玩家认输
4. **返回主菜单**: 放弃当前游戏，返回主菜单

## 技术实现

### 主要类说明

#### 1. Board类
- 管理棋盘状态
- 胜负判定算法
- 落子历史记录
- 撤销功能

#### 2. Player类（基类）
- HumanPlayer: 人类玩家
- AIPlayer: AI玩家，支持三种难度

#### 3. GameStatistics类
- 记录游戏统计信息
- 计算胜率、平局率
- 保存到JSON文件

#### 4. GameSaver类
- 游戏状态保存/加载
- 管理保存文件
- 支持多个保存点

#### 5. GomokuGame类
- 游戏主控制器
- 菜单系统管理
- 游戏流程控制

### AI算法说明

#### 简单难度
- 随机选择空位置落子

#### 中等难度
- 优先选择棋盘中心区域（5x5中心区域）
- 中心区域无空位时随机选择

#### 困难难度
1. **获胜检查**: 如果有立即获胜的位置，选择它
2. **防守检查**: 如果对手有立即获胜的位置，阻止它
3. **评估函数**: 根据以下因素评估位置得分：
   - 距离中心的位置（中心得分更高）
   - 潜在连线长度（二连、三连、四连）
   - 攻防平衡（进攻得分10倍，防守得分8倍）
